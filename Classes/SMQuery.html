Overview

SMQuery exposes an interface for defining a query against StackMob’s datastore API.

SMQuery uses StackMob’s terminology: – Queries are performed against a specific schema (usually also the name of a model class or of an entity in a managed object model) – Queries may select or order based on the values of the fields of an object belonging to their schema.
